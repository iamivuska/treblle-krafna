startScheduler(app)

getSchedulerConfig()

Reads scheduler config from DB under key meta:schedulerConfig
Returns object with: { cronExpression, description, paused }
Default if not set: { cronExpression: "0 9 1,15 * *", description: "1st and 15th of each month at 9am", paused: false }

setSchedulerConfig(cronExpression, description)

Saves new cron expression and description to DB
Restarts the cron job with the new schedule
